Чему вы научитесь

Программирование на Python

Python - это высокоуровневый и один из самых популярных языков программирования, который не знает только ленивый. В этот модуль войдут изучение списков, словарей, функций и классов.

HTML, CSS и JS

HTML, CSS и JS - это три кита, на которых держится все веб-программирование. Конечно существует множество модулей, облегчивших до невозможности разработку веб-приложений и множество языков программирования, которые выполняют все те же функции, что и HTML, CSS, и JS, но знать основы должен каждый веб-программист. Этот модуль включает в себя изучение основ разработки на HTML, CSS и JS

Программирование на Django

Django - высокоуровневый веб-фреймворк Python, который способствует быстрой разработке и его изучение облегчит жизнь веб-разработчикам. В этом модуле буду изучаться основные функции Django: создание приложений, страниц, связь между страницами, создание баз данных и управление ею, создание форм и т.п.
